Versions:
- python 3.8.10
- Django 4.2.4

OS:
- Zorin OS 16.3 (ubuntu 20.04)

Run:
```shell
    $ /usr/bin/env python ./manage.py makemigrations
    
    $ /usr/bin/env python ./manage.py migrate
    
    $ /usr/bin/env python ./manage.py runserver [ [localhost or address]:[port] ] # default 127.0.0.1:8000
```
